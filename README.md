# Luv Patel — Developer Portfolio

> A single-page portfolio for **Luv Rajeshbhai Patel** — Backend Software Engineer specializing in distributed workflow automation, Identity Governance &amp; Administration (IGA) platforms, and enterprise identity integrations.

Built as one self-contained HTML file — no build step, no frameworks, no dependencies beyond Google Fonts and Font Awesome.

## ✨ Highlights

- **Animated hero** — deep-navy theme with a subtle animated grid, a rotating typing effect, and a floating `engineer.java` code card
- **Light / dark theme toggle** — remembers your choice (`localStorage`), defaults to dark
- **Experience timeline** — click any highlight to expand full detail; includes education
- **Skills** — seven categories as hover-lift badge clusters
- **Projects** — featured full-stack app + algorithms projects with live links
- **Animated counters** — count-up stats triggered on scroll
- **Contact cards** — email, LinkedIn, GitHub, LeetCode, phone
- **UX polish** — scroll progress bar, scroll-spy navbar, smooth scroll, back-to-top, fade-in-on-scroll
- **Fully responsive** — mobile-first, with a hamburger menu
- **Accessible** — semantic HTML, `aria` attributes, keyboard-operable controls, and full `prefers-reduced-motion` support

## 🛠 Tech

Vanilla **HTML · CSS · JavaScript** — no framework, no bundler, no build.
Only external resources: [Google Fonts](https://fonts.google.com/) (Inter, Space Grotesk, JetBrains Mono) and [Font Awesome](https://fontawesome.com/) via CDN.

## 🚀 Run locally

No tooling required — just open the file:

```bash
open index.html        # macOS
# or double-click index.html in your file explorer
```

## ☁️ Deploy

This repo is a static site. On [Vercel](https://vercel.com):

1. **Import Git Repository** → select this repo
2. **Framework Preset:** Other · **Build Command:** _empty_ · **Output Directory:** _default_
3. **Deploy**

Every push to `main` then auto-deploys.
